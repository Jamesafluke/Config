## Install

- Rectangle (hotkey window snapping)
    - Configured my snap hotkeys to Control + Option +Up/Left/Right.
    - Command + H minimizes by default.
- Karabiner-Elements
    - Cmd Delete delete previous word.
    - Cmd Left/Right jump a word
- Razor Synapse
- Synergy
- AutoFocus (auto focuses whatever window your mouse hovers).
- Brew
- `brew install --cask iterm2`
    - Settings > Profiles > Keys > Key Bindings > Presets > Natural Text Editing.
    - Handles Option/Command navigation.
- `brew install neovim`
- `brew install git`
- `brew install terraform`
- `brew install azure-cli`
- `brew install --cask linearmouse`
- `brew install oh-my-posh`
    - `oh-my-posh font install`
    - JetBrainsMono
    - Select the font in Terminal settings.
    - `echo 'eval "$(oh-my-posh init zsh --config https://raw.githubusercontent.com/JanDeDobbeleer/oh-my-posh/main/themes/thecyberden.omp.json)"' >> ~/.zshrc` 

## Config
- Caps lock > escape
- Function lock off
- Keyboard Shortcuts > Function Keys.
- Turn off keyboard auto-correct (Keyboard > Input Sources > Edit
- Trackpad speed up
- Trackpad scroll speed
- Accessibility > Scroll Speed
- Turn off some Menu Bar icons
- Show file extensions
  - Finder > Settings > Advanced

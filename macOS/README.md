## iTerm
For Option Backspace: Settings > Profiles > Keys > General Left option key: Esc+
Add to ~/.zshrc
```
# Delete word backward (Option + Delete)
bindkey '^[^?' backward-kill-word

# Delete word forward (Option + Fn + Delete / Option + Forward Delete)
bindkey '^[[3~' kill-word
```
